<hl>

# Project Features
<hl>
  <ol type="i">
    <li>WelCome Page</li>
    <li>Login Page</li>
    <li>Home Page</li>
    <li>Registration Page</li>
    <li>All Team Information Page</li>
    <li>Rules Page</li>

<hl>
  <hl>
  <ol>
    <li>create a netbeans  project</li>
    <li>add  different jar files, SQL JDBC Driver</li>

</ol>
